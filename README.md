# Parcialito Vehículos: Comunicación de un diseño

## Enunciado

Dada [la siguiente solución (archivo Vehiculo.kt)](src/main/kotlin/ar/edu/unsam/algo2/logistica/Vehiculo.kt) se pide que

1. comunique el diagrama de clases
2. identifique cuál es el caso de uso principal y realice un diagrama de secuencia para una carga agrupada de un material peligroso y una caja estándar en una ruta de ripio
3. indique qué patrones de diseño aparecen y justifique qué valor agregado aportan
